Description

The experiment consists of selecting \(n\) balls at random, without replacement, from an urn with \(m\) balls, \(r\) of which are red and the other \(m - r\) green. Random variable \(Y\) gives the number of red balls in the sample, and is recorded on each run in the data table. This random variable has the hypergeometric distribution with parameters \( m \), \( r \), and \( n \).
