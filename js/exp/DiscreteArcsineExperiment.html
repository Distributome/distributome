Discrete Arcsine Distribution Experiment

Walk graph Distribution graph

Description

The experiment consists of tossing \(n\) fair coins. The position of the random walk after \(j\) tosses is the number of heads minus the number of tails. The random variables of interest is \( L \) the time of the last zero, which has the discrete arcsine distribution. The random walk is shown in red in the left graph on each update. A red dot is shown at \((L, 0)\). The value of \( L \) is recorded in the data table on each run. The probability density function and moments of \( L \) is shown in blue in the distribution graph and are recorded in the distribution table. On each run, the empirical density function and moments are shown in red in the distribution graph and are recorded in the distribution table. The number of steps \(n\) can be varied with the input control.
