Half-Normal Experiment

Brownian motion Distribution graph

Description

The experiment consists of running the standard Brownian motion process \( \boldsymbol{X} = \{X_s: s \in [0, \infty) \} \) on the interval \( [0, t] \). On each run, the path is shown in the graph on the left. The random variable of interest is the maximum position \( Y_t = \max\{X_s: s \in [0, t]\} \) which has the half-normal distribution with parameter \( \sqrt{t} \). On each run, the value of this variable is recorded in the first table, and the point \( (0, Y_t) \) is shown as a red dot in the path graph on the left. The probability density function and moments, and the empirical density function and moments, are shown in the distribution graph on the right and given in the distribution table on the right.
