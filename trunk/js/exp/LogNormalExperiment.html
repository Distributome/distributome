Lognormal Experiment

Brownian motion Distribution graph

Description

The experiment consists of running the geometric Brownian motion process \( \boldsymbol{V} = \{\exp(X_s): s \in [0, \infty) \} \) on the interval \( [0, t] \), where \( \boldsymbol{X} = \{X_s: s \in [0, \infty)\} \) is standard Brownian motion. On each run, the path is shown in the graph on the left. The random variable of interest is the final position \( \exp(X_t) \), which has the lognormal distribution with parameters 0 and \( \sqrt{t} \).

On each run, the value of the variable is recorded in the first table, and the point \( (t, \exp(X_t)) \) is shown as a red dot in the path graph on the left. The probability density function and moments, and the empirical density function and moments, are shown in the distribution graph on the right and given in the distribution table on the right. The parameter \( t \) can be varied with the input control.
