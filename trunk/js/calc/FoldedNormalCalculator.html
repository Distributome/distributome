Folded-Normal Distribution Calculator

Distribution graph

Description

This is a calculator for the folded-normal distribution with parameters \( \mu \) and \( \sigma \). The calculator gives the value of the cumulative distribution function \(p = F(x)\) for a given value of \(x\), or the value of the quantile function \(x = F^{-1}(p)\) for a given value of \(p\). The parameters \( \mu \) and \( \sigma \) can be varied with input controls, and the variables \(x\) and \(p\) can be varied with input controls.
