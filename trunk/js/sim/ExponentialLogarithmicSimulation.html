Exponential-Logarithmic Distribution Simulation

Distribution graph

Description

This applet simulates a value of a random variable with the exponential-logarithmic distribution on the interval. The value is recorded on each update. The shape parameter \(p\) and the scale parameter \( b \) can be varied with the input controls.
